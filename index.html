SLX FPS Booster


Hello, my name is SLX and I am proficient with Java programming of over 12 years. My enthusiast lies in the Minecraft's game code where I get to manipulate how it reacts to different exploits or glitches. My current project is the SLX FPS Booster mod riding on the Fabric API. I have been working on this project since January 2021. By reverse engineering Optifine's code, I managed to get an increase frame rate of 40% increase on an Intel HD 2000 Graphic.

    Stick Compass Positioner (Beta)

    Yes it is a weird name, but it is a stick that functions as a structure finder. You simply throw it in the water and it will show you where the structure is.




    Reworked NoCom Exploit (Buggy Alpha)

    This one is based on the NoCom exploit of 2b2t. I am reworking the code to where the AI can better predict every coordinates without relying on at least 80% of the server input. In other word, it is a coordinate exploit to find any player base positions. Spigot will be more buggy than PaperMC, expect crash.



    Ores Breadcrumbs

    Ability to follow breadcrumbs particles while traversing caverns to find ores. This will only work on surface ores to make it natural and undetectable by moderators. Features different particles from selections as well.



    Chat Bypass

    Exploits server code to send invisible messages among SLX's users. Will not guaranteed to work, try at your own risk.
