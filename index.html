SLX FPS Booster


Hello, my name is SLX and I am proficient with Java programming of over 12 years. My enthusiast lies in the Minecraft's game code where I get to manipulate how it reacts to different exploits or glitches. My current project is the SLX FPS Booster mod riding on the Fabric API. I have been working on this project since January 2021. By reverse engineering Optifine's code, I managed to get an increase frame rate of 40% increase on an Intel HD 2000 Graphic.


Testing Phase


The server that I am currently interested in is desertsmp.tk. So far I deployed 4 Beta tester for around 2 weeks and one has gotten banned. I will continue to keep testing on this server discretely since it provided good results in improving my mod.





Fabric API

This mod supports the latest version of Fabric API up to 1.17.1 from 1.16.5!




Stick Compass Positioner

Yes it is a weird name, but it is a stick that functions as a structure finder. You simply throw it in the water and it will show you where the structure is.




Reworked NoCom Exploit

This one is based on the NoCom exploit of 2b2t. I am reworking the code to where the AI can better predict every coordinates without relying on at least 80% of the server input. In other word, it is a coordinate exploit to find any player base positions.
